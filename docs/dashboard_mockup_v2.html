Besloten — Startpagina
Cockpit is het beginscherm. Eén klik naar alles.
Vervangt de stap-voor-stap workflow. Na inloggen ziet Martien direct zijn actieve orders met deadlines, EV-waarschuwingen en quick-actions per order.
Cockpit = startscherm
Besloten — Type Z en C
Z = Boogzinker (BZ). C = Calculatie (alleen Sigma).
Workflow B/N/Z is gelijk, alleen boogzinker-profiel is anders. Type C is een apart product: alleen Sigma berekeningen, geen tekening of werkplan. Vooral voor provincie Groningen.
Z = Boogzinker C = Calculatie
Besloten — Wij zijn de bron
Platform vervangt GSheets. Export en backup altijd mogelijk.
Architectuurbeslissing Inodus. Eén bron = meer flexibiliteit voor statusmail, facturatie, werkplan generator. GSheets wordt archief na migratie.
Platform = source of truth
Besloten — Tekenaar per order
Default = Martien, aanpasbaar per order.
Martien reviewt en accordeert alle ontwerpen. Nid (Sopa Choychod) houdt de orderlijst bij en doet uitwerkingen. Later uitbreidbaar met meer tekenaars.
Default ML, wijzigbaar
Besloten — Migratie alles
Volledige GSheets migratie. Wat niet compleet is vult Martien aan.
Geen schaduwperiode — volledige import. Onduidelijke velden kan Martien later aanvullen in het platform. GSheets blijft als archief-backup.
